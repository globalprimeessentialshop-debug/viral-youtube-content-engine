## 🧠 What This Does

Create **5 viral YouTube videos automatically** with AI agents that:

1. **🧠 Master Brain Agent** - Analyzes trends, makes decisions, learns 24/7
2. **✍️ Script Agent** - Writes viral scripts with hooks
3. **🎬 Video Agent** - Creates production specifications
4. **📱 Social Media Agent** - Designs thumbnails & titles
5. **🚀 Distributor Agent** - Plans growth & distribution

All working together with **YOUR approval at each step**.

## ⚡ Quick Start

### Installation

```bash
# Clone repository
git clone https://github.com/globalprimeessentialshop-debug/viral-youtube-content-engine.git
cd viral-youtube-content-engine

# Install dependencies
pip install -r requirements.txt
pip install -r jarvis_requirements.txt
```

### Start Using It

**Option 1: Interactive Mode (Easiest)**
```bash
python jarvis_integration.py
# Type: Jarvis
```

**Option 2: Web Dashboard (Visual)**
```bash
python dashboard.py
# Open: http://localhost:8000
```

**Option 3: Command Line (Fast)**
```bash
python jarvis_cli.py start
```

## 🐛 Troubleshooting

### Dashboard not loading?
```bash
# Make sure Flask is installed
pip install flask flask-cors

# Check port 8000 is available
# Start dashboard
python dashboard.py
```

### Agents not executing?
```bash
# Check API keys in .env
# Verify all dependencies installed
pip install -r requirements.txt
pip install -r jarvis_requirements.txt

# Check logs for errors
```
